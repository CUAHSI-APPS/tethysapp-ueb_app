Welcome to UEB Model Services!

Utha Energy Balance model is a snowmelt model developed for the prediction of rapid snowmelt rates potentially responsible for soil erosion and surface water inputs. See more details
This application will provide different services to help the user automatically prepare a complete model input package for a user-defined research domain (Within US continent).
This application will also execute the UEB model using cloud computing resources for a complete model input package provided by the user.
