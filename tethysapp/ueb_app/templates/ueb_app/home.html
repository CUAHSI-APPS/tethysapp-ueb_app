Welcome to UEB Model Services!

The Utah Energy Balance model is a snowmelt model developed for the prediction of snow accumulation and melt, driven by inputs of precipitation, wind, humidity, temperature
and radiation at sub diurnal time scales. See more details
This application provides services to prepare a complete UEB model input package for a user defined area (within the bounds of supporting data in the continental US)
The application also provides services to submit a UEB model package for execution on a server or high performance computing center installation.
